Application: Calculator with Google Cloud
Version: 1.0.0

To run Application, you need to follow the instruction below.

Server Side
1. Open backend folder and run command "yarn install".
2. Start server by running command "yarn start" in Backend folder to start server.

Client Side
1. In Frontend folder, run command "yarn install" to download dependencies.
2. Run command "yarn run build-app" to build application.
3. Open calculator.exe in calculator-win32-x64 folder.
