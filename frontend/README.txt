APPLICATION : Calculator
Release version: 1.0.0
IMPORTANT NOTE: You need to start server before run application.

/*-----------------FOR DEVELOPER-----------------*/
To initiate application for developer, please setup as following:
1. run command "yarn install".
2. After installation, run command "yarn start".

/*-----------------BUILD APPLICATION-----------------*/
To build application, please do as following:
1. run command "yarn install".
2. command "yarn run build-app".
After finished, you will get folder name "calculator".
You can open application from calculator.exe inside the folder.
